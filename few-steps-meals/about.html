About Few Steps Meals

A simple, practical cooking project created to help busy people, students, missionaries, and families make delicious meals without stress or complicated ingredients.

Hi, I'm Merari Flores

I created Few Steps Meals as part of my WDD 131 course, but also as a personal project inspired by real life — tight budgets, busy schedules, and the need for meals that are fast, affordable, and actually taste good.

I love cooking simple dishes that don’t require fancy tools or expensive ingredients. My goal is to help others feel confident in the kitchen, even if they’re just starting out or cooking on a budget.

Why This Project Exists

Few Steps Meals was built to support people who need:

  • ✔ Simple recipes with clear steps
  • ✔ Budget-friendly ingredients
  • ✔ Quick meals for busy days
  • ✔ Easy meal ideas when they don’t know what to cook
  • ✔ A clean, friendly website that’s easy to navigate

This project also demonstrates my skills in HTML, CSS, JavaScript, responsive design, accessibility, and dynamic content creation.

Built for WDD 131

This website was created as the final project for WDD 131 – Dynamic Web Fundamentals. It includes:

  • ✔ Valid, accessible HTML
  • ✔ Clean, responsive CSS
  • ✔ JavaScript features including DOM interaction, events, arrays, objects, and localStorage
  • ✔ A recipe filter system
  • ✔ A meal idea generator
  • ✔ A consistent design system across all pages

Everything was built from scratch — no templates, no frameworks, and no third‑party builders.
